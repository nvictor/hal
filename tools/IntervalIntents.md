# Perceived Communicative Interval Intents

Perceived communicative intent of directed intervals (Western tonal context)

0 → 1 — Tension
1 → 0 — Doubt
0 → 2, 2 → 0 — Compliance
0 → 3 — Sadness
3 → 0 — Call
0 → 5 — Unfinished
0 → 6 — Unstable
0 → 7 — Assertive (Question when unresolved)
7 → 0 — Agreement
0 → 8, 0 → 9 — Wonder
0 → 10, 0 → 11 — Insistence
0 → 12 — Expansion (Transcendence)

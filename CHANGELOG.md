# HFN Changelog

All notable changes to the Harmonic Flow Notation (HFN) specification will be documented in this file.

## [1.0.0] - 2026-02-16

### Added
- Initial formalized specification.
- Center-state operators (=, ⇄, ⊃, ↗, →).
- Chord motion operators (=, ⇄, →).
- Octave modifiers (↑, ↓).
- Section-based syntax.
